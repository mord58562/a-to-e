## 4. Hard rules — banned tokens (audit before commit)

- **Em-dashes (U+2014):** ZERO tolerated. Use ` - ` (space-hyphen-space) instead. Audit by `grep -P "\\x{2014}" file.json | wc -l`. If any found, replace and re-audit.
- **"ATSI":** the abbreviation is permanently banned. Spell out "Aboriginal and Torres Strait Islander" in full for general references. When a specific person is named in a stem, use only the applicable half: "Aboriginal" OR "Torres Strait Islander" (never both unless general). Applies to stems, options, rationales, explanations, sources, tags, subtopics.
- **"canonical":** permanently banned word in any output. Use "primary", "core", "main", "live", or "promoted" instead.
- **Em-dash audit method:** never trust visual reading. Count U+2014 bytes directly with grep.

## 5. Self-audit pass before commit

For every batch:
1. Valid JSON. Validate with `json.load`.
2. Re-read every rationale; check every numeric claim, dose, threshold, age cut-off, GA cut-off against the cited source. Drop unverifiable claims.
3. **Difficulty distribution targets (REVISED 2026-06-01, OVERHAUL v3):** the bank now uses 5 difficulty tiers (1-5/5). Every existing question was shifted down 1 level on 2026-06-01 (former 5/5 -> 4/5, 4/5 -> 3/5, 3/5 -> 2/5, 2/5 -> 1/5). A new 5/5 tier exists ABOVE the prior top - reserved for the hardest items.
   - **L1 (1/5):** single-fact recall or one-step application. <=5% of any batch. Reserve for fundamentals that every Y4 student should default-answer.
   - **L2 (2/5):** integrate 2-3 data points and choose between PLAUSIBLE distractors. Lead-in is "next step", not "what is it". 30-60 sec real reasoning. ~50% baseline.
   - **L3 (3/5):** multi-step pathway with trade-offs, OR application of an AU-specific cut-off / threshold / dose-rule with competing considerations. 60-90 sec.
   - **L4 (4/5):** atypical or contested presentation, anti-pattern recognition, OR niche sub-specialty fact PLUS multi-step reasoning. Niche fact alone is NEVER L4. **Target 15-20% of bank.** 90-150 sec.
   - **L5 (5/5):** L4 plus one or more of: calculation under uncertainty, a disconfirming finding that kills the textbook default, competing guidelines requiring you to choose one cut-off, a multi-branch algorithm where the wrong branch is genuinely tempting. **~5% target.** Niche topic alone is NEVER enough for L5.
   For 15-Q batches aim ~1 L1 / 6 L2 / 5 L3 / 2 L4 / 1 L5.
   For 30-Q batches aim ~1 L1 / 12 L2 / 10 L3 / 5 L4 / 2 L5.
   For 50-Q batches aim ~2 L1 / 22 L2 / 17 L3 / 7 L4 / 2-3 L5.
3a. **Difficulty = the complexity of the clinical reasoning required to land on the correct answer.** Stem length, number of data fields, vocabulary, and topic rarity are NOT primary difficulty drivers.
3b. **>=50% of L4-L5 must centre on COMMON presentations** (sore throat, PPH, ACS, paracetamol overdose, postnatal depression, asthma exacerbation) where reasoning depth EARNS the rating - not rare-topic recognition.
3c. **Never write L0** (no question, telegraphed answer, or single-step recall of a memorised fact phrased trivially). If a draft can only reach that level, discard it.
4. (Reserved.) Stem-length floors per difficulty have been REMOVED. Write the stem to be as long or as short as the clinical scenario genuinely requires; never pad to hit a word target, never truncate to hit one. **DO add paragraph breaks** when a stem exceeds ~80 words so it reads as scannable clinical prose, not a wall of text. The only stem-length constraint that remains is "every sentence must be justified" - see item 11.
5. Each option's rationale ends with explicit source attribution by name (e.g. "(Source: RCH CPG: Bronchiolitis)").
6. Each option has a `source_refs` array of strings matching labels in the question's top-level `sources`.
7. 2+ entries in top-level `sources`, Australian first.
8. Correct-letter distribution: not all-A; ideally ~3 per letter for 15-Q batch, ~4 per letter for 20-Q batch.
9. Option length-balance: all five options within ±25% word count of each other. Never write a noticeably longer correct answer.
10. Distractor hooks in stem (critical for ≥3/5): each of the five distractors must be supported by at least one specific feature in the stem.
11. Every stem sentence justified: (a) hooks specific named distractor, (b) points to correct answer, (c) intentional red herring (max 1), or (d) essential context. Delete anything not justified.
12. No weight-based dose arithmetic in lead-in. Stem MAY mention a dose to anchor; correct answer must NEVER hinge on student doing mg/kg multiplication.
13. AU SI units only (mmol/L, micromol/L, g/L, x10^9/L, mmHg, °C, kg).
14. AU spellings (paediatric, gynaecology, haematology, haemorrhage, foetal, oesophagus, oedema, anaemia, leukaemia, orthopaedic, fibre, colour, behaviour, signalling, caesarean, dyspnoea, diarrhoea, faeces). Exception: drug INN "magnesium sulfate" follows AMH spelling.
15. Mandatory `model` field on every question = `"Claude Opus 4.7 (1M context)"` if you are Opus 4.7, or your specific model+version otherwise. Never leave blank or write "unknown".
16. No uni-specific framing (no UNE, JMP, MEDI6101, course codes, specific faculty names).
17. No banned tokens (em-dash, "ATSI", "canonical").
18. **No author meta-commentary in the stem.** Stems must read like clinical vignettes the candidate would see in an exam. Grep generated batches against these patterns and strip any matches:
    - `\(this hooks` and `hooks the [^)]*distractor`
    - `rules out [A-E]` (when used to describe distractor rejection)
    - `red herring\)`, `tests the X anti-pattern`, `this anchors the`
    Distractor-hook reasoning belongs in the rationale, not the stem.
19. **Tags policy.** Each question carries at most 3 tags in `tags[]`. Reuse existing high-frequency tags - do NOT invent new specific tags. Guideline-source names (RCH CPG, RANZCOG, eTG, AMH etc.) go in `sources`, NOT in `tags`. Generic modifiers (severe, complex, common, rare, first-line, acute, chronic) are NOT tags. Pick tags from the live vocabulary (pregnancy, contraception, pre-eclampsia, SSRI, ultrasound, DKA, asthma, bronchiolitis, anaphylaxis, clozapine, etc.). If you need a new tag, it must apply to >=3 questions to earn its place; otherwise omit.
20. **No diagnosis-leak through option text.** When the stem deliberately withholds the diagnosis to test diagnostic + management reasoning together, NO option text may name that diagnosis - not the correct option, not any distractor. Options describe ACTIONS (e.g. "IV ceftriaxone with aggressive sepsis resuscitation") not diagnoses (NOT "IV ceftriaxone for suspected meningococcal meningitis"). The rationale carries the diagnosis post-reveal. Exception: lead-ins of the form "what is the most likely diagnosis?" - obviously the options ARE the diagnoses there. Audit pattern to grep options for: `for suspected `, `because .* confirmation`, `since .* is more likely`, `to rule out `, `consistent with `, plus diagnosis nouns appearing in only one option that aren't in the stem/lead-in.

21. **Answer-length parity (hard rule, 2026-05-21).** The correct option MUST NOT be (a) noticeably longer than the longest distractor, (b) the only one with a key qualifier (e.g. "with airway clearance physiotherapy", "no contraindications"), (c) the only one with a numeric value, or (d) uniquely comprehensive. Pad distractors to equal depth/specificity OR shorten the correct option. Compute character lengths across options; if max/min > 1.4, rewrite.

22. **Pertinent-negative parsimony.** Do NOT include contraindications/qualifiers inside the correct option text (e.g. "ergometrine (no hypertension contraindication) plus TXA"). The CI clears the option AND telegraphs the answer. Such qualifiers belong in the rationale. In the stem itself, list only pertinent negatives that a real clinician would document - withholding some is itself a reasoning test.

23. **User is ALWAYS the doctor.** Every stem positions the candidate as the treating doctor (intern / RMO / registrar / consultant as appropriate, but always a doctor). Do NOT write stems where the discriminator is "you are not a doctor and cannot prescribe", or where the candidate is the practice nurse / midwife / pharmacist / paramedic / student observer being asked to defer. Anything other than a doctor-perspective is a waste of an exam slot. If the clinical scenario has another professional administering screening (e.g. child-health nurse runs the EPDS), that's fine as scene-setting, but the QUESTION is what the doctor receiving the referral does next.

24. **No lead-in guideline leaks.** The lead-in must NOT name the guideline being tested ("best matches RCH and ISPAD paediatric DKA guidance"). Use neutral lead-ins: "What is the best initial fluid plan?". The guideline-name belongs in rationales/sources only.

25. **No "follow local guidelines" non-options.** Any option text that reduces to "do what the local guideline says" / "as per institutional policy" trivializes by elimination. Replace with a substantive distractor.

26. **No literal markdown asterisks.** Stems and lead-ins must NOT contain `**...**` - those are LLM formatting artefacts. Plain text only.

27. **Stem's last sentence must not telegraph the answer.** Hard demote-to-discard pattern (2026-06-01): the final sentence of the stem must NOT describe the intervention, manoeuvre, or management plan that the correct option uses. E.g. "the mother consented antenatally to internal manoeuvres and breech extraction for a non-cephalic second twin" telegraphs internal podalic version as the answer. Use neutral end-of-stem language: "the team has been briefed and the mother had a thorough antenatal discussion of intrapartum options".

28. **No "offer all options and support patient choice" correct answers** when there is no genuine clinical discriminator. Lead-in must commit: "What would you FIRST recommend...?" The rationale acknowledges shared decision-making but the correct option names a specific defensible first recommendation.

29. **Lay-language for patient/family reports** (audit before commit). When the patient/family reports a finding, use lay language ("dark coloured urine", "long thin lump in his right upper tummy", "starts crying suddenly, draws his legs up, then settles"). Reserve medical English for CLINICIAN-observed findings (auscultation, imaging interpretation, exam descriptions). Pattern grep: "currant-jelly stool", "sausage-shaped mass", "colicky abdominal pain", "bilious vomiting" - stem-level auto-reject unless lead-in is "most likely diagnosis".

30. **Numeric not adjectival.** "BP 165/110", not "severely hypertensive". "Myometrium 2 mm", not "thin myometrium". The candidate is expected to know the threshold.

31. **Disconfirming finding for L3+.** Every L3+ question should contain at least one finding that rules out the textbook default differential, forcing the candidate to reason past it. Examples: "DAT negative, blood groups matched" kills HDN; "previous Hb >110 three years ago" kills inherited thalassaemia despite Mediterranean heritage; "renal function and electrolytes within normal limits" kills uraemia/SIADH.

32. **Vitals fixed order.** When vitals are listed in prose, use comma-separated single sentence in fixed order: pulse rate / blood pressure / respiratory rate / SpO2 / temperature (then foetal HR if obstetric). Either give all five OR compress to "vital signs within normal limits". Do NOT mix.

33. **data_table values are FLAT STRINGS.** Do NOT nest dicts in data_table values (e.g. `"ultrasound_findings": { "EFW": "...", "AC": "..." }` is wrong). Either flatten to a single comma-separated string ("EFW <3rd centile; AC <3rd centile; UA Doppler AEDF") OR split into multiple flat keys ("Estimated foetal weight": "...", "Abdominal circumference": "..."). The renderer is defensive but the schema is flat-only.

34. **Reference ranges mandatory.** Every load-bearing lab gets its range in parentheses on the same line: "haemoglobin 90 g/L (120-160)". A naked number is decorative-only - if the lab IS load-bearing and you forgot the range, it's a leak. Omit decorative labs entirely.

35. **bhCG threshold for ectopic surgical management (RANZCOG GPS):** the threshold for REQUIRING surgical management of tubal ectopic is >=5000 IU/L (not 3000). Methotrexate is preferred when bhCG <5000 AND other RANZCOG criteria are met (no foetal cardiac activity, mass <3.5 cm, no significant pain, no haemorrhage, reliable follow-up, no contraindications). Cite RANZCOG by full name in the rationale.

36. **RCH Melbourne CPG is the authoritative paediatrics source.** When AU paediatric guidance varies (e.g. UTI imaging - RCH says imaging only for atypical UTI or recurrence or infants <6 months without antenatal monitoring; NOT routine for typical UTI under 24 months), follow RCH. Do not cite KHA-CARI alongside RCH where they diverge.

37. **RCH-specific cut-offs and protocols** (from the 2026-06-01 audit; 61 prior questions diverged from these):
    - **UTI imaging cascade:** No routine renal ultrasound after a single typical UTI even under 24 months. US during illness only for atypical UTI (poor response to 48 h Abx, non-E coli, sepsis, abdominal mass, raised creatinine). Infants under 6 months: US for recurrent UTI or no antenatal monitoring. **DMSA and MCUG are NOT in the RCH cascade** - they are NICE constructs. Do NOT make DMSA the correct answer if citing RCH.
    - **Bronchiolitis:** First-line oxygen is LOW-FLOW (nasal prong 1-2 L/min). HFNC reserved for failure of low-flow after 2 hours. SpO2 target is 90% (not 92-94%).
    - **Croup:** Dexamethasone 0.15 mg/kg orally for MILD or MODERATE croup; 0.6 mg/kg ONLY for severe. Observation 3 hours minimum after last nebulised adrenaline (not 4). Nebulised adrenaline is a FLAT 5 mL of 1:1000, not per-kg.
    - **Anaphylaxis:** Adrenaline IM dose by banded age/weight table (ASCIA). EpiPen Jr 150 mcg for 10-20 kg; EpiPen 300 mcg for 20+ kg. Do NOT use 10 mcg/kg arithmetic when prescribing - banded table is the AU standard.
    - **Paediatric sepsis pathway:** Order is IV/IO access -> blood cultures -> first IV antibiotic (within the hour) -> 20 mL/kg saline bolus (over 5-10 min, repeat to 60 mL/kg total). Inotropes/adrenaline AFTER fluids, not before.
    - **Fever under 3 months:** Full septic workup including LP for any unwell-appearing infant or under-28-day-old; empirical IV cefotaxime (or per local sepsis pathway).
    - **Gastroenteritis dehydration:** NGT rehydration is FIRST-LINE for mild-moderate dehydration that fails oral; IV reserved for severe dehydration, shock, or NGT failure. Single dose ondansetron 0.15 mg/kg for vomiting (under 8: 2 mg; 8-15 kg: 2 mg; 15-30 kg: 4 mg).

## 8. Sources — Australian first

## 9b. Pause sentinel

Before doing anything else on a run:

```
if git ls-files --error-unmatch .pause-remote-agent 2>/dev/null; then
  echo "PAUSED: .pause-remote-agent present in repo root. Exiting without generating."
  exit 0
fi
```

If `.pause-remote-agent` exists in the repo root (tracked file), the routine exits immediately without generating, committing, or pushing. This lets the maintainer pause the routine from any machine with one commit:

```
touch .pause-remote-agent && git add .pause-remote-agent && \
git commit -m "Pause remote routine" && git push origin main
```

To resume: delete the file and push.

This is the preferred pause mechanism (no need to call the API). The maintainer should still also flip `enabled: false` on the routine via the dashboard if pausing for >24h, to save Anthropic credits.
